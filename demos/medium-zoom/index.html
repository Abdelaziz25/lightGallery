Medium like image zoom

Add a zoom feature to your blog images just like on Medium. With lightGallery's mediumZoom plugin, it's easy to give your readers a closer look at your pictures, straight away

If you want to target all images in a blog post, you can add lightGallery to the outer element and select all images using the selector option.

HTML structure# 
<div class="medium-zoom-demo">
    <p>Lorem ipsum dolor sit amet......</p>
    <figure
        lg-background-color="white"
        class="blog-images"
        data-src="image-1.jpg"
        data-lg-size="1600-1126"
    >
        <img src="thumbnail-1.jpg" />
    </figure>
    <p>Lorem ipsum dolor sit amet......</p>
    <p>Lorem ipsum dolor sit amet......</p>
    <p>Lorem ipsum dolor sit amet......</p>
    <figure
        lg-background-color="rgb(28 62 74)"
        class="blog-images"
        data-src="image-2.jpg"
        data-lg-size="1600-1126"
    >
        <img src="thumbnail-2.jpg" />
    </figure>
    <p>Lorem ipsum dolor sit amet......</p>
    <figure class="blog-images" data-src="image-3.jpg" data-lg-size="1600-1126">
        <img src="thumbnail-3.jpg" />
    </figure>
</div>
JavaScript# 
lightGallery(document.querySelector('.medium-zoom-demo'), {
    // Target all images
    selector: '.blog-images',
    // Add medium zoom plugin
    plugins: [lgMediumZoom],
});
