Commercial License#

lightGallery is a free and open-source library, however, if you are using the library for business, commercial sites, projects, and applications, the Commercial license is the appropriate license. With this option, your source code is kept proprietary.

Once purchased, you’ll receive a commercial license PDF and be all set to use lightGallery in your commercial applications.

Extended Commercial license#

If you want to include lightGallery as part of software developer kit (SDK), web application builder or website builder, downloadable or installable products like Wordpress themes, HTML templates,or something that produces copies that each use lightGallery, you need to choose the Extended Commercial license

Open source license#

If you are creating an open source application under a license compatible with the GNU GPL license v3, you may use this project under the terms of the GPLv3. Questions? Read the GPL FAQ.
